# Instrucciones para Codex

## Contexto

Este repositorio implementa el CRM corporativo integrado al portal existente PortalCorporativo.

## Implementacion

El CRM se implementa como modulo integrado y reutilizable.

## Reglas de trabajo

- Usar seguridad del portal.
- Usar auditoria del portal.
- Usar notificaciones del portal.
- Usar archivos del portal.
- Usar menus del portal.
- Usar temas del portal.
- Usar catalogos globales del portal.
- Mantener configuracion visual parametrizable.
- Mantener Kafka o RabbitMQ para una fase futura.
- Manejar credenciales mediante configuracion segura.
- Conectar Salesforce o Dynamics solo mediante CRM Integration Hub.

## Antes de modificar

Indicar agente responsable, fase, archivos a crear, archivos a modificar, dependencias y criterios de aceptacion.

## Despues de modificar

Actualizar estado de agentes, dependencias, asuntos abiertos y decisiones en docs/coordination.
