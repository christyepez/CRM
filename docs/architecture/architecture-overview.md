# Arquitectura CRM integrada al Portal Corporativo

## Contexto

PortalCorporativo define la plataforma transversal. Este repositorio CRM implementa el modulo funcional CRM y su Integration Hub.

## Diagrama logico

Portal Angular, CRM Module, CRM API, CRM Core, CRM Integrations, Portal APIs y CRMs externos.

## Principio

El CRM se integra con el portal; no lo reemplaza ni lo duplica.
