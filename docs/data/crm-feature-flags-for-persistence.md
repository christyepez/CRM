# CRM Feature Flags for Persistence

Proposed flags for future activation:

- `CRM_PERSISTENCE_ENABLED=false`
- `CRM_FOUNDATION_MODE=true`
- `CRM_PRODUCTIVE_CRUD_ENABLED=false`
- `CRM_PORTAL_AUTH_REQUIRED=true`

These are design flags only in P1. They are not wired to runtime configuration yet and no `.env` file is introduced.
