# CRM Durable Persistence Readiness Checklist

Required before real DB activation:

- Shared SQL Server strategy approved.
- CRM logical database ownership approved.
- Migration naming and rollback approved.
- Connection and secret management approved.
- Backup and restore plan approved.
- Data retention and PII policy approved.
- Test data policy approved.
- Observability and health checks approved.
- Feature flag rollout approved.

Current P5 state: `databaseReady=false` and `durablePersistence=false`.
