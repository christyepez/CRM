# Modelo de Datos CRM

## Entidades CRM

Customer, Contact, Lead, Opportunity, Activity, Case, Campaign, Interaction, Note, CrmDocument, CrmTag y CrmAssignment.

## Campos obligatorios

Las entidades principales deben incluir identificador, tenant, compania, fechas de creacion y actualizacion, usuarios de creacion y actualizacion, eliminacion logica y estado activo.
