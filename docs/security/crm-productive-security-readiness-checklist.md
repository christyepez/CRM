# CRM Productive Security Readiness Checklist

Required before productive Auth or CRUD:

- Portal-owned Auth runtime approved.
- Tenant/user context contract approved.
- Permission runtime adapter approved.
- Audit events and correlation approved.
- No CRM-owned login, Identity, JWT or cookie auth.
- Secrets management approved.
- PII handling and retention approved.
- Rollback and incident response defined.
